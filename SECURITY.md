# Security Policy

## Supported Versions

This portfolio website is actively maintained. The latest version on the `main` branch is supported.

| Version | Supported |
|---------|-----------|
| Latest  | ✅ |

## Reporting a Vulnerability

If you discover a security issue or vulnerability related to this website, please report it privately.

**Email:** hello@ysgstudio.in

or

**Contact:** https://ysgstudio.in/contact.html

Please include:
- A description of the issue
- Steps to reproduce it
- Screenshots (if applicable)

We aim to acknowledge reports within 5 business days and resolve confirmed issues as quickly as possible.

Thank you for helping keep this website secure.
